# Руководство по работе с приглашениями в группы

## Как отправить приглашение

1. Войдите в систему как владелец или модератор группы
2. Перейдите на страницу группы (Groups → выберите группу)
3. В разделе "Участники" нажмите кнопку "Пригласить пользователя"
4. Введите email пользователя, которого хотите пригласить
5. Нажмите "Отправить приглашение"

**Технические детали:**
- Приглашение отправляется через API: `POST /api/v1/notify/invoke/{groupId}?invokeByUsername={вашEmail}`
- В теле запроса передается: `{ username: "email_пользователя", expiresAt: "дата_истечения" }`
- Срок действия приглашения: 7 дней с момента создания

## Как принять приглашение

1. Войдите в систему как пользователь, которому пришло приглашение
2. Перейдите на страницу "Приглашения" (Notifications) в меню
3. В разделе "Новые приглашения" вы увидите список приглашений со статусом PENDING
4. Для каждого приглашения вы можете:
   - Нажать "Принять" - статус изменится на ACCEPTED
   - Нажать "Отклонить" - статус изменится на DECLINED

**Важно:** После принятия приглашения вы НЕ добавляетесь автоматически в группу. Владелец или модератор группы должен вручную добавить вас в группу.

## Как добавить пользователя в группу после принятия приглашения

1. Войдите как владелец или модератор группы
2. Перейдите на страницу группы
3. В разделе "Участники" вы увидите секцию "Принятые приглашения (требуют добавления в группу)"
4. Для каждого принятого приглашения нажмите "Добавить в группу"
5. Пользователь будет добавлен в группу со статусом MEMBER

**Технические детали:**
- API для добавления: `PATCH /api/v1/group/{groupId}/user-id/{userId}`
- Пользователь добавляется автоматически в статусе MEMBER

## Статусы приглашений

- **PENDING** - Ожидает ответа от пользователя
- **ACCEPTED** - Пользователь принял приглашение
- **DECLINED** - Пользователь отклонил приглашение
- **EXPIRED** - Приглашение истекло по сроку действия
- **CANCELLED** - Приглашение отменено

## Важные замечания

1. Приглашения можно отправлять только владельцам и модераторам группы
2. Пользователь должен быть зарегистрирован в системе (его email должен существовать)
3. После принятия приглашения владелец/модератор получает уведомление о необходимости добавить пользователя
4. Если пользователь не найден в списке при добавлении, обновите список пользователей
